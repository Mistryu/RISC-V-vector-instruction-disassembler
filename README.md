# RISC-V Vector Instruction Disassembler

A small Python script that converts RISC-V Vector (RVV) instructions into human-readable assembly.

## Usage

```bash
python3 rvv_disassembler.py <instruction>
```

Both hex and integer inputs are allowed

### Example

```bash
python3 rvv_disassembler.py 0x02110557
```
